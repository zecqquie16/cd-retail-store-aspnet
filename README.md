# Online CD Retail Store  
### Agile Development Team Project

This project is a multi-project **C# / ASP.NET** solution developed as part of an **Agile team coursework**.  
The objective was to design and implement an **online CD retail store**, combining a structured application architecture with a **relational SQL Server database** using **T-SQL tables and stored procedures**.

The system focuses on clean separation between application logic and data access, and demonstrates collaborative development within an iterative Agile workflow.

---

## Project Objectives
- Design a simple online retail system for managing music CDs
- Implement a relational database with structured tables and stored procedures
- Apply Agile development principles in a team environment
- Develop a modular C# solution with reusable components
- Practice database-driven application development

---

## Tech Stack
- **C#**
- **ASP.NET**
- **SQL Server**
- **T-SQL (tables & stored procedures)**
- **Visual Studio**
- **Git & GitHub** (team collaboration)
